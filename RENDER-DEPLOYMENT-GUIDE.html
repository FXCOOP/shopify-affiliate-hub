‚ú® OPTION 1: Use Blueprint (EASIEST - RECOMMENDED)

This is the fastest and easiest way to deploy. Just a few clicks!

Step 1: Go to Render Dashboard

Go to: https://dashboard.render.com

Step 2: Click "New +" Button

Look for the blue "New +" button in the top right corner

[Look for: Blue button labeled "New +" in top right]

Step 3: Select "Blueprint"

From the dropdown menu, click "Blueprint"

[Click: Blueprint option from the dropdown]

Step 4: Connect Your Repository

Select: FXCOOP/shopify-affiliate-hub

Render will scan the repository and find automation-system/render.yaml

Step 5: Review Services

Render will show you 2 services it will create:

  • shopify-automation-api (Web Service)
  • shopify-automation-daily (Cron Job)

Step 6: Click "Apply Blueprint"

Click the "Apply" button at the bottom

What Happens Now?

Render will automatically create both services for you!

Wait 2-3 minutes for deployment to complete.

Step 7: Add Environment Variables

After deployment, you need to add your API keys to BOTH services.

IMPORTANT: Add to BOTH Services

You must add the OPENAI_API_KEY to both services:

  • shopify-automation-api
  • shopify-automation-daily

For each service:

  1. Click on the service name
  2. Go to Environment tab
  3. Click Add Environment Variable
  4. Add this REQUIRED variable:
Key: OPENAI_API_KEY Value: sk-proj-[your-actual-openai-api-key]

How to Get Your OpenAI API Key

1. Go to: https://platform.openai.com/api-keys

2. Click "Create new secret key"

3. Copy the entire key (starts with sk-proj-)

4. Paste it into Render

Step 8: Redeploy Both Services

After adding the API key:

  1. Click "Manual Deploy" ‚Üí "Deploy latest commit"
  2. Do this for BOTH services
  3. Wait for deployment to complete (2-3 minutes)

‚úÖ Done!

Your automation system is now live!

Dashboard URL: https://shopify-automation-api.onrender.com/backoffice

üîß OPTION 2: Create Services Manually

If the Blueprint doesn't work, you can create each service manually:

A. Create Web Service (API + Dashboard)

1. Click "New +" ‚Üí "Web Service"

2. Fill in these settings:

Name: shopify-automation-api Repository: FXCOOP/shopify-affiliate-hub Branch: main Root Directory: automation-system Runtime: Node Build Command: npm install Start Command: node api-server.js Plan: Free

3. Add Environment Variables:

OPENAI_API_KEY = sk-proj-[your-key] OPENAI_MODEL = gpt-4o-mini SITE_URL = https://shopify-success-hub.onrender.com AFFILIATE_LINK = https://shopify.pxf.io/Qj5oDz PORT = 3000 LOG_LEVEL = info

4. Click "Create Web Service"

B. Create Cron Job (Daily Automation)

1. Click "New +" ‚Üí "Cron Job"

2. Fill in these settings:

Name: shopify-automation-daily Repository: FXCOOP/shopify-affiliate-hub Branch: main Root Directory: automation-system Runtime: Node Build Command: npm install Command: node main.js --now Schedule: 0 7 * * * Plan: Free

3. Add the SAME Environment Variables as above

4. Click "Create Cron Job"

üÜì OPTION 3: Use GitHub Actions (100% FREE)

If you don't want to use Render at all, use GitHub Actions instead:

Why GitHub Actions?

  • ‚úÖ Completely FREE (2,000 minutes/month)
  • ‚úÖ Already configured in your repository
  • ‚úÖ Runs daily at 7 AM automatically
  • ‚úÖ No additional services needed

Step 1: Go to Your Repository

https://github.com/FXCOOP/shopify-affiliate-hub

Step 2: Add Secret

1. Click Settings (top menu)

2. Click Secrets and variables ‚Üí Actions

3. Click New repository secret

4. Add:

Name: OPENAI_API_KEY Value: sk-proj-[your-actual-key]

5. Click Add secret

Step 3: Enable Actions (if needed)

1. Click Actions tab

2. If disabled, click I understand my workflows, go ahead and enable them

Step 4: Test It

1. Go to Actions tab

2. Click Daily Shopify Automation

3. Click Run workflow

4. Watch it run!

GitHub Actions Limitation

With GitHub Actions, you won't have a live dashboard. For the dashboard, you still need Render's Web Service.

Best of both worlds: Use GitHub Actions for automation + Render Web Service for dashboard!

‚ùì Troubleshooting

Blueprint Button Not Working?

Use Option 2 (Manual Setup) instead. It's the same result, just a few more clicks.

OpenAI API Error 401?

Your API key is invalid. Check:

  • Is billing enabled on OpenAI?
  • Did you copy the entire key?
  • Is the key still active?

Services Not Deploying?

Check the Logs tab in Render to see what went wrong.

Need Help?

Run the health check locally:

cd automation-system node health-check.js

This will tell you exactly what's missing!
